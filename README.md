# 🏴‍☠️ Will of Data
### *One Piece Grid Logic & XAI Predictor*

**Will of Data** è un sistema di IA progettato per risolvere griglie logiche basate sull'universo di *One Piece*. Il progetto integra modelli di **Machine Learning** e un motore di **Scoring Ibrido** per bilanciare coerenza narrativa e rarità statistica.

---

## Obiettivo
Sviluppo di un sistema di Machine Learning in grado di risolvere una griglia di attributi (One Piece Grid Game) ottimizzando la scelta del personaggio in base a criteri di pertinenza, rarità statistica e importanza narrativa.

## Pipeline di ML
1. **Data Engineering**: Gestione missing values e feature encoding.
2. **Preprocessing**: Bilanciamento tramite Undersampling.
3. **Modeling**: Classificatori Random Forest per la predizione degli attributi.
4. **Scoring Engine**: Algoritmo di ranking basato su Distanza Probabilistica e Rarity Boost.

## Core Features
* **Grid Solver:** Risoluzione di griglie 3x3 (Fazioni, Haki, Razze, Ruoli).
* **Hybrid Scoring:** Mix tra probabilità ML, **Hard Constraints** e **Rarity Boost**.
* **Audit System (XAI):** Interfaccia per ispezionare le decisioni dell'IA e il *Ground Truth*.
